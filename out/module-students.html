(inner) /students/query(students, data, operator, value) → {Array.<string>}

[POST] Obtain all students that satisfy the query specifications.
Parameters:
Name Type Description
students Array.<string> The student codes of the students which the query will be applied. Dont add this parameter if you want to apply the query to all the students.
data string The name of the attribute that will be checked in the query students.
operator string The logical operator that will be used in the query. The possible values are: '<','<=','=','>=','>','<>'.
value float The value used for the comparison.
